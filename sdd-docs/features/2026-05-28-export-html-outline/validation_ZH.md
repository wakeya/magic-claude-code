# Validation

## 功能验证清单

### 基础验证

- [ ] 导出的 HTML 文件无外部依赖（检查 `<head>` 中无 `src=` 引用外部资源）
- [ ] 文件可离线打开（双击即可在浏览器中加载）

### 大纲面板验证

- [ ] 大纲条目数量与 user 消息数量一致
- [ ] 每个条目的预览文本正确（最多 50 字符，超过截断 + `...`）
- [ ] 每个条目显示时间戳（格式 `YYYY-MM-DD HH:mm:ss`）

### 交互验证

- [ ] 点击大纲条目 → 页面平滑滚动到对应消息
- [ ] 滚动页面 → 当前可见消息的大纲条目自动高亮
- [ ] 回到顶部按钮 → 页面平滑滚动到顶部
- [ ] 小屏（<1024px）浮动按钮可见且可点击
- [ ] 小屏点击浮动按钮 → 弹窗正常打开
- [ ] 小屏弹窗中点击大纲条目 → 正常跳转且弹窗关闭

### 响应式验证

- [ ] 1024px 及以上宽度：右侧面板固定显示
- [ ] 1023px 及以下宽度：右侧面板隐藏，显示浮动按钮
- [ ] 窗口大小切换时布局正确更新

### 主题验证

- [ ] 亮色主题（`data-theme="light"`）：大纲面板样式正确
- [ ] 暗色主题（`data-theme="dark"`）：大纲面板样式正确

### 性能验证

- [ ] 文件大小增长 <20KB
- [ ] 滚动时 IntersectionObserver 无明显性能问题

### 回归验证

- [ ] 现有导出功能不受影响（header、main 消息、system/tool 折叠等原有功能正常）
