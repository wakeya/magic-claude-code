## 概要

在使用统计页面新增紧凑快捷日期范围条，提供 `今日`、`近7天`、`近30天`，并同步现有开始日期和结束日期筛选。

`今日` 包含今天。`近7天` 和 `近30天` 不包含今天，只覆盖今天之前已经完整结束的自然日。

## 已确认

1. 快捷日期条放在使用统计二级页签和筛选条件之间。
2. UI 使用紧凑一行工具条，不使用高卡片。
3. `今日` 包含今天。
4. `近7天` 和 `近30天` 不包含今天。
5. 默认选中 `近7天`。
6. 手动日期匹配预设时高亮，不匹配时不高亮。
7. 复用现有 `from`、`to`、`tz` 查询参数，不新增后端范围参数。
