# Claude Code 会话记录浏览器验证

**功能：** Claude Code 会话记录浏览器
**状态：** draft
**最后更新：** 2026-05-20

## 验收标准

1. 会话从 `~/.claude/projects/` 读取，无需修改代理逻辑。
2. 项目作为主要导航列出，带会话数量。
3. 选择项目后展示该项目的会话，按最近活跃排序。
4. 会话详情按时间顺序展示消息，角色正确。
5. 用户消息高亮并列在大纲侧边栏中。
6. 系统提示和工具结果默认折叠。
7. 点击大纲项滚动到对应消息。
8. HTML 导出生成可在离线环境打开的自包含 `.html` 文件。
9. 管理界面不删除 JSONL 文件或附属目录；清理入口仅展示 Claude Code CLI 命令提示。
10. Agent 会话文件（`agent-*`）不在列表中显示。
11. 损坏或空的 JSONL 文件不会导致应用崩溃。

## 手动验证

1. 确认 `~/.claude/projects/` 可访问（或在 Docker 中挂载）。
2. 启动代理/管理服务。
3. 打开管理界面，导航到 `会话记录` 标签页。
4. 验证项目列表展示 `~/.claude/projects/` 下的项目。
5. 选择一个项目，验证会话显示正确的标题。
6. 打开会话，验证消息按顺序显示。
7. 验证用户消息高亮并出现在大纲中。
8. 点击大纲项，验证滚动定位。
9. 点击"导出"，验证下载的 HTML 在浏览器中正确打开。
10. 打开清理提示，验证展示 `claude project purge --dry-run <project-path>` 和交互式终端清理建议，且文件未被管理面板删除。
11. 验证代理继续正常工作（未修改代理代码）。
