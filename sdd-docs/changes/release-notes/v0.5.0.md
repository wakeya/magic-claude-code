## v0.5.0 更新内容

### Added
- **供应商限流队列**：支持按 Provider 配置最大并发数、排队等待和超时，防止超出上游 API 并发限制
- **429 自动重试**：可配置的指数退避重试策略，自动解析 Retry-After 头部，支持 1302/1305 错误码识别
- **启动消息国际化**：服务启动提示支持中英文自动检测
- 前端供应商弹窗新增限流配置区域，支持中英文

### Changed
- CI 发布说明优先从 `sdd-docs/changes/release-notes/vX.Y.Z.md` 读取，替代硬编码文本

### Docs
- 在 AGENT.md 和 CLAUDE.md 中补充提交与发布工作流指南

## v0.5.0 Release Notes

### Added
- **Provider rate-limit queue**: Per-provider configurable max concurrency, queueing, and timeout to stay within upstream API concurrency limits
- **429 auto-retry**: Configurable exponential-backoff retry that parses the `Retry-After` header and recognizes 1302/1305 error codes
- **Startup-message i18n**: Service startup messages auto-detect Chinese/English
- Frontend provider modal adds a rate-limit config section (i18n)

### Changed
- CI release notes are now read preferentially from `sdd-docs/changes/release-notes/vX.Y.Z.md`, replacing hardcoded text

### Docs
- Added commit-and-release workflow guide in AGENT.md and CLAUDE.md
