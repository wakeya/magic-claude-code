## v0.9.3 更新内容

### Added
- **全局回到顶部悬浮按钮**：原来仅在会话记录 tab 存在的回到顶部按钮提升为全局功能，所有 tab 页共享
  - **触发条件**：页面滚动超过 100px 时在右下角显示，避免微小滚动时闪烁
  - **交互**：点击平滑滚动到页面顶部（`window.scrollTo({ top: 0, behavior: 'smooth' })`）
  - **层级**：`z-30`，低于 outline modal（`z-40`）和 cleanup modal（`z-50`），打开弹窗时不干扰交互
  - **自动隐藏**：短内容 tab（如证书信息）内容不超出视口时按钮自动隐藏
  - **事件清理**：`scroll` 监听在 `onBeforeUnmount` 中清理，无内存泄漏
  - **SessionBrowser 清理**：移除 `SessionBrowser` 中已冗余的固定按钮，保留 outline modal 内部的 back-to-top

## v0.9.3 Release Notes

### Added
- **Global back-to-top floating button**: The back-to-top button previously only on the session-records tab is promoted to a global feature shared by all tabs
  - **Trigger**: Shown in the bottom-right corner after scrolling past 100px, avoiding flicker on minor scrolls
  - **Interaction**: Click smooth-scrolls to the top (`window.scrollTo({ top: 0, behavior: 'smooth' })`)
  - **Z-index**: `z-30`, below the outline modal (`z-40`) and the cleanup modal (`z-50`), so opening a modal isn't disrupted
  - **Auto-hide**: On short-content tabs (e.g. certificate info) where content doesn't overflow the viewport, the button auto-hides
  - **Event cleanup**: The `scroll` listener is cleared in `onBeforeUnmount`, with no memory leaks
  - **SessionBrowser cleanup**: Removed the now-redundant fixed button in `SessionBrowser`; the back-to-top inside the outline modal is retained
